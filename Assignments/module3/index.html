What is...?

HTML

Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.(Back to Top)

CSS

Cascading Style Sheet, are used to format the layout of Web pages. They can be used to define text styles, table sizes, and other aspects of Web pages that previously could only be defined in a page's HTML.(Back to Top)

JavaScript

an object-oriented computer programming language commonly used to create interactive effects within web browsers.(Back to top)
